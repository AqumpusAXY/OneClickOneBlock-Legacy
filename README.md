# 点到为止:遗产

---

## 描述

**点到为止遗产 (1C1BL)** 是 [点到为止(1C1B)](https://github.com/LucunJi/OneClickOneBlock) 的一个**非官方向后移植**版本，专门为 Minecraft Forge 1.12.2 版本设计。

## 功能特点

- ✅ 在玩家破坏方块后添加可配置的短暂延迟
- ⌨️ 提供热键来切换功能的开启/关闭
- ⚙️ 允许通过配置文件自定义延迟持续时间

## 致谢

- 特别感谢 [LucunJi](https://github.com/LucunJi) 创造了原模组

# One Click One Block Legacy

---

## Description

**One Click One Block Legacy (1C1BL)** is an **unofficial backport** of [One Click One Block (1C1B)](https://github.com/LucunJi/OneClickOneBlock) designed specifically for Minecraft Forge version 1.12.2.

## Features

- ✅ Adds a configurable short delay after the player destroys a block
- ⌨️ Provides a hotkey to toggle the functionality on/off
- ⚙️ Allows customization of the delay duration through configuration
